## Getting Started
  
### Prerequisites

* Node 16.10
* Angular CLI
  ```sh
  npm install -g @angular/cli
  ```
### Installation

1. Install project dependencies
   ```sh
   npm install
   ```
2. Run the project
   ```sh
   npm start
   ```
